# Llaves de prueba

Sube cualquiera de estos archivos `.key` en `/unlock` para entrar con un usuario distinto.
No hay base de datos todavía: cada llave es independiente y tu progreso se guarda en el
`localStorage` del navegador donde la subiste. Desde Ajustes puedes descargar tu propia
llave actualizada (con tu progreso, monedas, nivel, etc.) para moverla a otro navegador.

- **admin-full.key** — rol admin, acceso completo a todos los cursos.
- **demo-course.key** — acceso solo al "Curso Demo: Primeros pasos" (`course-demo`).
- **estudiante-course-001.key** — acceso solo al curso "Domina Google NotebookLM" (`course-001`).
